Key Advantages

Built for modern cloud entry control and harsh environments

This anti-vandal terminal combines a tempered glass panel, metal housing, multiple access methods, and cloud-ready connectivity. It is suitable for smart buildings, shared entry points, and integrated access control scenarios.

Ethernet / Wi-Fi / Bluetooth QR / IC Card / Password / Touch Key HTTP / MQTT / RS485 / Relay Output

Core Features

Multiple Access Methods

Supports IC card, QR code, touch password, touch key, and Bluetooth access workflows.

Dynamic QR Support

Compatible with forward and reverse dynamic QR codes for flexible visitor and cloud access use cases.

Platform Integration

Supports third-party platform integration through HTTP, MQTT, RS485 input, and relay output.

Durable Construction

Designed with tempered glass and CNC metal housing for stronger protection in public entry areas.
