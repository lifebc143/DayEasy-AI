Key Advantages

Ultra-thin reader for smart credentials and platform integration

This reader combines QR code scanning, Bluetooth access, multiple card technologies, and compact wall-mounted installation in a thin 86 x 86 x 7 mm design. It is suitable for projects that need mobile integration, app access, or SDK-based deployment.

QR / Bluetooth / IC / CPU / ID / NFC Mini Program / App / SDK Ready Fast Scan Under 0.2 Seconds

Core Features

Multi-Credential Support

Supports QR codes, Bluetooth, IC cards, CPU cards, ID cards, and NFC for flexible access control projects.

Integration Friendly

Supports WeChat Mini Program access, app-based use cases, and SDK integration for custom systems.

Fast and Accurate Reading

Reads in less than 0.2 seconds at a scan distance of 5 to 15 cm with wide reading and scanning angles.

Features a 9H tempered glass panel, metal housing, and a thin 7 mm profile for clean installation.
